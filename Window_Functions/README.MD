*** Aggregate & Window Functions

Today’s SQL learning focused on mastering Aggregate Functions and Window Functions — two powerful tools for data analysis and reporting.

This repository contains practice queries demonstrating how to summarize data and perform advanced calculations across result sets without losing row-level detail.

 Topics Covered
🔹 Aggregate Functions

Used to perform calculations on groups of rows and return a single summarized value.

✔ COUNT() — count records
✔ SUM() — total values
✔ AVG() — average value
✔ MIN() — smallest value
✔ MAX() — largest value
✔ GROUP BY — grouping data
✔ HAVING — filtering grouped results

🔹 Window Functions

Used to perform calculations across a set of rows related to the current row — without collapsing the result set.

✔ OVER() clause
✔ PARTITION BY
✔ ORDER BY within windows
✔ Running totals
✔ Ranking functions
✔ Row-wise comparisons

** Key Learning Outcomes

- Understand difference between GROUP BY vs Window Functions
- Perform calculations while preserving row-level data
- Generate rankings and running totals
- Analyze data trends efficiently
- Write cleaner analytical SQL queries

** Why This Matters

Aggregate functions summarize data.
Window functions analyze data.

Together, they unlock powerful real-world analytics like:

- Sales trends
- Ranking systems
- Performance comparisons
- Running metrics
